# 🧠 ReComunity Website

A web platform for **KMUTNB Robotic Engineering** designed to enhance the learning experience and foster collaboration among students and educators.

---

## 🚀 Getting Started

This project uses the following tech stack:

- [Next.js (App Router)](https://nextjs.org/docs/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/docs/)
- [DaisyUI](https://daisyui.com/docs/)

### 📦 Installation

```bash
pnpm install
pnpm dev
```

---

## 🎨 UI Design

### Font

- `Roboto Mono` – clean, modern, and readable.

### Theme

- ☀️ Light Mode: `bumblebee`
- 🌙 Dark Mode: `dim`

### Color Variables

These custom CSS variables are used to control the theme appearance via DaisyUI:

```css
--color-base-100
--color-base-200
--color-base-300
--color-base-content
--color-primary
--color-primary-content
--color-secondary
--color-secondary-content
--color-accent
--color-accent-content
--color-neutral
--color-neutral-content
--color-info
--color-info-content
--color-success
--color-success-content
--color-warning
--color-warning-content
--color-error
--color-error-content
```
